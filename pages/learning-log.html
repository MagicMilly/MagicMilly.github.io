2021-02-25

- Command to find a file and its path located anywhere: find . type -f | grep FILENAME
- To find a directory, change to find . type -d | grep DIRNAME
- Command to open a file in a specific application: open -a "APPLICATION" FILENAME. To open this file without a sublime alias, I used open -a "Sublime Text" learning-log.html while in the directory where this is located.

2020-11-11

- To escape from Sublime's "distraction free" view, moving your mouse to the top of the screen will show the toolbar.
- Do not let "perfect" be the enemy of "good", especially when spending 30 minutes to decide on just the right color for a font.
- You cannot create your own wiki using GitHub pages, as they only host static content.
